Interoperability

[ ] Decide on standard datatype source.
Standard datatypes are in two places, $CX_ROOT/src/data/datatypes.cex and in functions such as cx_surface_create_datatypes(), which must agree with eachother. There should be only authoritative source for standard datatypes. Some choices might be:

  • Use $CX_ROOT/src/data/datatypes.cex exclusively.

  • Divide up $CX_ROOT/src/data/datatypes.cex into sections, e.g. molecule-dts.cex, surface-dts.cex, camera-dts.cex, etc., and use catcex to insert them in a stream.

  • Create functions such as cx_surface_create_datatypes() for all packages and require their use.

[ ] Decide how to register datatypes and implement solution.
By champion, committee, automatic database?
